# Cyber Security Daily Ops

Interactive cybersecurity operations platform designed to simulate a real-world defensive security environment, demonstrating day-to-day activities across vulnerability management, Microsoft Defender XDR, EDR, SAST, DAST, Cloud Security, SRE collaboration, patch management, incident response and security operations.

The project was developed as a practical portfolio demonstration focused on defensive security, cyber risk analysis, monitoring, threat containment and communication between Security, Governance, Development, Cloud, SRE and Red Team teams.

## Main Features

* Animated cybersecurity operations timeline
* Simulated Microsoft Defender XDR / EDR alert flow
* Vulnerability management workflow
* SAST and DAST validation process
* Cloud and SRE collaboration simulation
* Patch management lifecycle
* MITRE ATT&CK mapping
* NIST and CIS Controls alignment
* Incident response and containment flow
* Executive security reporting simulation
* Interactive SOC dashboard
* GitHub Pages deployment ready

## Project Goal

The main objective of this project is to visually demonstrate how a cybersecurity professional operates in a modern defensive security environment, covering:

* Vulnerability identification and prioritization
* Threat containment
* Security monitoring
* Cloud and SRE integration
* Risk analysis
* Patch management
* Security maturity improvement
* Red Team collaboration
* Executive communication
